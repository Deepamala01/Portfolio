# Deepamala Chockalingam - PCB Design Engineer Portfolio

A professional single-page portfolio website for a Junior PCB Design Engineer, featuring a modern dark theme with orange accents, responsive design, and interactive elements.

## Features
- **Hero Section**: Animated introduction with glowing profile effect.
- **Skills**: Dynamic progress bars for technical competencies.
- **Projects**: Detailed cards for PCB design projects (Altium Designer).
- **Experience**: Timeline view of professional history.
- **Responsive**: Fully optimized for Desktop, Tablet, and Mobile.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- JavaScript (Intersection Observer, Smooth Scroll)
